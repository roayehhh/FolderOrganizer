# FileSorter

Hi!  
This project sorts files in a folder based on their extension.

## How to use

1. Run the Python script `filesorter.py`.
2. Enter the path of the folder you want to organize.
3. The script will create folders named after file extensions (e.g., Txt, Pdf) inside the given folder.
4. Files will be moved to their respective extension folders.

## Requirements

- Python 3.x
- Modules used: `os`, `shutil` (built-in, no need to install)
